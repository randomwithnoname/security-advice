# Software You Should Use If You Care About Privacy and Security

I will add information like I did with the software I recommend avoiding soon.

## App Stores
- [Accrescent](./accrescent.md)
- Google Play Store ([see issues with Aurora Store](./aurora-store.md))
- [AppVerifier](./appverifier.md) + [Obtainium](./obtainium.md)

## Operating Systems
- [GrapheneOS](https://privsec.dev/posts/android/choosing-your-android-based-operating-system/#grapheneos)
- [DivestOS](https://privsec.dev/posts/android/choosing-your-android-based-operating-system/#divestos) is a great choice if you've already got a device it supports and have no other option.
- iOS with lockdown mode (if it doesn't affect usability too much) or the stock Google Pixel OS are the only operating systems you should buy new devices for if [GrapheneOS isn't an option](https://privsec.dev/posts/android/banking-applications-compatibility-with-grapheneos/#introduction). Remove and opt out of privacy-invasive services when you can.
