[SimpleX](https://simplex.chat/) is a messaging app which has the same robust security guarantees as Signal but doesn't require a phone nunber or email address to sign up. 

It will do a better job at protrcting data technologically rather than relying on [a single organization with a good track record](https://signal.org/bigbrother/). Not using email addresses or phone numbers for account registration is a great start, but it goes beyond that. SimpleX doesn't use long-term identifiers for sending or receiving messages, which means you can delete your address and all your contacts and groups remain. This prevents spam and stalking. SimpleX also uses different servers for sending and receiving messages, which will make it very difficult for a malicious server operator to store metadata on users once there are multiple preset server operators, assuming they don't collude. Regardless, SimpleX has full support for using [Tor](https://en.m.wikipedia.org/wiki/Tor_(network)). It allows dual server addresses for when servers are accessible via Tor hidden services (.onion), and has support for using a SOCKS5 proxy in the app, so you can route traffic through Tor without needing a system-wide VPN service for that.
